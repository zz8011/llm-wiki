# Seedance2-Storyboard-Generator

**GitHub**: https://github.com/liangdabiao/Seedance2-Storyboard-Generator
**Stars**: 789 | **Forks**: 145
**语言**: Claude Code Skill

## 项目概述

基于 Claude Code + Skill + Seedance 2.0 的完整 AI 视频制作工作流，专注于将小说、故事转化为多集视频系列。

## 工具三件套

1. **Claude Code** - 剧本创作与分镜脚本生成
2. **Nana Banana Pro** - 角色、场景、道具素材图生成
3. **Seedance 2.0** - 视频生成与延长

## 工作流程

```
构思主题 → 写剧本 → 生成素材描述 → 生图 → 写分镜脚本 → 逐集生成视频
```

## 素材编号规范

- C01-C99：角色
- S01-S99：场景
- P01-P99：道具
