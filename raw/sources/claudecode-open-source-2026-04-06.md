# ClaudeCode Open Source

**GitHub**: https://github.com/soongenwong/claudecode
**Stars**: 679 | **Forks**: 278
**语言**: Rust

## 项目概述

"A high-performance, open-source AI coding agent written in Rust."

独立的开源实现，受 Claude Code 启发，非 Anthropic 官方产品。

## 核心特性

- Rust 驱动：内存安全、最小二进制、高执行速度
- Agentic CLI：交互式 shell
- 模型灵活：支持 Anthropic/OpenAI 兼容 provider
- Workspace 感知

## 安装

```bash
# 从源码编译
cargo build --release
```
