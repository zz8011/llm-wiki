# 老张-Karpathy-Obsidian教程 - 来源摘要

## 概述

老张解读 Karpathy 的 LLM Wiki 方法，手把手教在 Obsidian 里落地。

## 核心内容

### 为什么 RAG 不够用？
- 每次提问 AI 都在从头搜寻知识
- 什么都没沉淀下来

### 实战步骤

1. **素材采集** - Obsidian Web Clipper 插件
2. **图片本地化** - Ctrl+Shift+D 下载图片
3. **图谱视图** - 发现孤立页面和知识盲区
4. **Git 版本管理** - Obsidian Git 插件自动 commit + push
5. **搜索** - qmd 本地搜索引擎

## 精髓

> 人类负责选素材、定方向、问好问题、思考意义，AI 负责其他一切。
