# ClaudeCode Open Source - 来源摘要

## 项目概述

用 Rust 编写的开源 Claude Code 实现，679 stars。

## 核心特性

- **Rust 驱动**：内存安全、最小二进制、高执行速度
- **Agentic CLI**：交互式 shell，支持单次 prompt
- **模型灵活**：支持 Anthropic/OpenAI 兼容 provider，还有 xAI/Grok 别名
- **Workspace 感知**：理解开发环境

## 背景

- 2026年3月31日 Claude Code npm source map 泄漏
- 社区开始分析和开源重建
- 韩国小哥用 Python/Rust 爆改，3天近15万星
