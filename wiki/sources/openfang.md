# OpenFang - 来源摘要

## 项目概述

开源 Agent 操作系统，Rust 编写，16.4k stars。

## 核心特点

- **One binary** - 一键部署
- **Battle-tested** - 生产可用
- **137K LOC, 14 crates** - 代码量庞大
- **1,767+ tests** - 测试完善
- **Zero clippy warnings** - 代码质量高

## 版本

v0.3.30 — Security Hardening Release (March 2026)

## 与 OpenClaw 关系

OpenFang 是 OpenClaw 生态的一部分，支持 MCP 协议。
