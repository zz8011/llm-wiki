# Seedance2-Storyboard-Generator - 来源摘要

## 项目概述

基于 Claude Code + Skill + Seedance 2.0 的完整 AI 视频制作工作流，789 stars。

## 工具三件套

| 工具 | 用途 |
|------|------|
| Claude Code | 剧本创作与分镜脚本生成 |
| Nana Banana Pro | 角色、场景、道具素材图生成 |
| Seedance 2.0 | 视频生成与延长 |

## 工作流程

```
构思主题 → 写剧本 → 生成素材描述 → 生图 → 写分镜脚本 → 逐集生成视频
```

## 项目特色

- 四幕结构剧本（起承转合）
- 素材编号规范：C角色/S场景/P道具
- Seedance 2.0 时间轴格式提示词
- 示例项目：林冲、聂风、鲁智深等
