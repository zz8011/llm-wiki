# 设计系统

AI 时代的设计系统构建方法。

## 核心理念

用 design.md 文件作为设计系统的核心，让 AI 代理能够理解和使用设计规范。

## 工作流

```
Stitch 出设计 → design.md 统一设计系统 → Claude Code 写代码
```

## 工具链

- **Google Stitch** - AI 设计工具
- **design.md** - 设计系统文档格式
- **Claude Code** - 根据 design.md 编写代码
- **MCP** - 连接设计和代码的桥梁

## 优势

- 以前请设计师要花几千美金等几周
- 现在一个人一个下午搞定
- 设计系统以 Markdown 格式存储，AI 可直接理解和引用
